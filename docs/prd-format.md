# PRD Format

## Required

ใช้ checkbox task แบบ GitHub Markdown:

```md
- [x] Done task
- [ ] Pending task
```

## Project Name

```md
# Project: Customer CRM
```

หรือ:

```md
# Customer CRM
```

## Sections

Heading ระดับ 2-6 จะถูกใช้เป็น section ของ task:

```md
## Backend
- [x] Create API
- [ ] Add tests
```

## Optional Weight

ถ้า task ใหญ่กว่าปกติ ใส่ weight ได้:

```md
- [ ] Payment integration <!-- weight: 5 -->
```
