# Contributing

## Workflow

1. Read `AGENTS.md`.
2. Check `PRD.md` for active tasks.
3. Make focused changes.
4. Run relevant tests.
5. Update documentation if behavior changes.
6. Update `PRD.md` task status.

## Commit Guidance

- Keep commits focused.
- Do not commit `.env`, database files, caches, `node_modules`, or build output.
- Explain meaningful behavior changes in the commit message.
